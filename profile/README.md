## ü§ñ Osprey Narrator

**AI-powered SAR narrative generation.** A fine-tuned LLM that transforms Osprey's raw alert output into structured, analyst-ready compliance narratives ‚Äî replacing hours of manual report writing with seconds of inference.

- **Instant Narratives** ‚Äî Feed it an Osprey alert JSON, get a complete SAR-style report
- **12 FATF Rules + 6 Typologies** ‚Äî Full coverage of AML/CFT compliance framework
- **Run Anywhere** ‚Äî Available as LoRA adapter (HuggingFace) or quantized GGUF (Ollama)
- **Trained on Synthetic Data** ‚Äî No real customer data used, fully reproducible

> *From alert to narrative in seconds, not hours.*
